# 书贝
基于Android的共享图书漂流软件
# 简介
在 整个APP 开发过程中使用了Android Studio开发平台，开发语言为 Java，还包含 Android SDK、MySQL、豆瓣图书 API 和百度地图 SDK 等多个辅助工具，最终实现了书友在线上分享交流图书漂流信息、线下直接借阅。
